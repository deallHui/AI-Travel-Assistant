## 🚀 部署步骤

### 1. 准备云服务器
```bash
# 购买云服务器（Ubuntu 20.04 LTS）
# 配置安全组，开放端口：80, 443, 8000
# 安装Docker和Docker Compose
```

### 2. 创建部署配置
```yaml
# docker-compose.yml
version: '3.8'
services:
  rag-ai:
    build: .
    ports:
      - "8000:8000"
    environment:
      - DEEPSEEK_API_KEY=${DEEPSEEK_API_KEY}
    volumes:
      - ./vectorstores:/app/vectorstores
      - ./travel_guides_database.txt:/app/travel_guides_database.txt
    restart: unless-stopped
  
  nginx:
    image: nginx:alpine
    ports:
      - "80:80"
      - "443:443"
    volumes:
      - ./nginx.conf:/etc/nginx/nginx.conf
      - ./ssl:/etc/nginx/ssl
    depends_on:
      - rag-ai
    restart: unless-stopped
```

### 3. 配置HTTPS
```bash
# 申请SSL证书（Let's Encrypt免费）
sudo apt install certbot
sudo certbot certonly --standalone -d your-domain.com
```

### 4. 部署应用
```bash
# 上传代码到服务器
git clone your-repo
cd rag-ai

# 构建和启动
docker-compose up -d

# 检查状态
docker-compose ps
```

## 🔧 配置优化

### Nginx配置
```nginx
server {
    listen 443 ssl;
    server_name your-domain.com;
    
    ssl_certificate /etc/nginx/ssl/cert.pem;
    ssl_certificate_key /etc/nginx/ssl/key.pem;
    
    location / {
        proxy_pass http://rag-ai:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

### 环境变量
```bash
# .env
DEEPSEEK_API_KEY=your_api_key
ENVIRONMENT=production
LOG_LEVEL=INFO
```

## 🔄 CI/CD自动部署

### GitHub Actions
```yaml
name: Deploy to Server
on:
  push:
    branches: [main]
jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - name: Deploy to server
        run: |
          ssh user@server "cd /app && git pull && docker-compose up -d --build"
```
