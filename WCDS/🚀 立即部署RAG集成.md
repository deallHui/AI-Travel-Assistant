## 🆘 如果遇到问题

### 常见问题解决
1. **云函数部署失败**
   - 确保网络连接正常
   - 重新运行 npm install
   - 检查微信开发者工具版本

2. **仍然返回固定回答**
   - 确认云函数已成功部署到云端
   - 检查RAG AI后端服务是否运行
   - 查看云函数调用日志

3. **网络连接错误**
   - 确认RAG AI后端运行在 http://localhost:8000
   - 检查防火墙设置
   - 验证健康检查接口

### 快速诊断命令
```bash
# 检查RAG AI后端状态
curl http://localhost:8000/wechat/health

# 测试查询接口
curl -X POST http://localhost:8000/wechat/query \
  -H "Content-Type: application/json" \
  -d '{"question":"推荐北京景点","openid":"test"}'
```
