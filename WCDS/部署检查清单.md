## 🚀 部署步骤

### 1. 准备RAG AI后端
```bash
cd rag_ai
python start_backend.py
```

### 2. 配置云函数
```bash
cd WCDS/cloudfunctions/getQASystem
npm install
```

### 3. 部署云函数
- 在微信开发者工具中右键点击 `getQASystem`
- 选择"上传并部署：云端安装依赖"
- 等待部署完成

### 4. 测试连接
```bash
cd WCDS
node test_rag_connection.js
```

### 5. 小程序测试
- 在微信开发者工具中预览小程序
- 测试AI助手功能
- 验证所有功能正常

## 🔍 故障排除

### 常见问题及解决方案

#### 1. 云函数调用失败
**症状：** 小程序显示"网络连接失败"
**检查：**
- [ ] 云函数是否正确部署
- [ ] 云函数日志是否有错误
- [ ] 网络权限是否正确配置

**解决：**
- 重新部署云函数
- 检查云函数代码语法
- 确认云开发环境配置

#### 2. RAG AI连接失败
**症状：** 云函数日志显示连接RAG AI失败
**检查：**
- [ ] RAG AI后端是否运行
- [ ] API地址配置是否正确
- [ ] 防火墙是否阻止连接

**解决：**
- 启动RAG AI后端服务
- 修正API地址配置
- 配置网络访问权限

#### 3. 回答质量问题
**症状：** AI回答不相关或质量差
**检查：**
- [ ] 向量数据库是否包含相关数据
- [ ] DeepSeek API是否正常工作
- [ ] 问题是否在支持范围内

**解决：**
- 重建向量数据库
- 检查API密钥配置
- 优化问题表述

#### 4. 界面显示问题
**症状：** 页面布局异常或样式错误
**检查：**
- [ ] WXSS文件是否正确加载
- [ ] 组件引用是否正确
- [ ] 数据绑定是否正常

**解决：**
- 检查样式文件语法
- 验证组件配置
- 调试数据流
